# ClawSentinel — Website

**Marketing and documentation website for ClawSentinel.**

Live at: [clawsentinel.sh](https://clawsentinel.sh)

---

## What This Repo Contains

The public-facing website for ClawSentinel — the active security layer for OpenClaw.

- Landing page with product overview
- Feature breakdowns for all 5 modules
- Installation guide
- Security documentation
- Competitor comparison
- Pricing page

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 14 (App Router) |
| Styling | Tailwind CSS |
| Deployment | Vercel |
| Analytics | Plausible (privacy-first) |

---

## Development

```bash
git clone https://github.com/hshdevhub/clawsentinel-web
cd clawsentinel-web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

---

## Deployment

Auto-deploys to Vercel on push to `main`.

```bash
git push origin main   # triggers Vercel deploy automatically
```
